Plugin generates graphical statistics of user's twitter follower/following counts to admin dashboard.

== Description ==

This plugin generates graphical charts using Google Charts API of user's Twitter follower / following and listed counts. Developed by [LumoLink](http://lumolink.com/).

Check out [plugin's homepage](https://github.com/LumoLink/wp-twtstats)

Authors:

* [LumoLink](http://www.lumolink.com)
* [@incd](http://twitter.com/incd)
* [@artemd](http://twitter.com/artemd)

Contributes: [@wickux](http://twitter.com/wickux)

== Installation ==

1. Upload wp-twitterstats to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set your username from plugin's settings.
